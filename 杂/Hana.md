### 2 什么是HANA？

High‐Performance ANalytic Appliance
定位：软硬件一体化设备
软件：内存计算引擎 / 内存数据库( SAP In‐Memory Database, IMDB )
硬件：认证的预配置的硬件系统( HP / CISCO / IBM / DELL / FUJITSU / HITACHI )
( 注意：没有SUN哦 )

行存储：每行数据放一起，适合交易型应用（OLTP）

列存储列存储：：每列数据放每列数据放 起一起，，适合分析型应用适合分析型应用（OLAP）
